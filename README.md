# ScanPort

Port Scanner desenvolvido em Python com foco em estudos de redes, automação e fundamentos de cibersegurança.

A ideia do projeto foi entender na prática como funciona a identificação de portas abertas e serviços expostos, aplicando conceitos utilizados em ferramentas de varredura de rede.

## Funcionalidades

- Varredura de portas TCP
- Identificação de portas abertas
- Execução com multithreading
- Interface simples via terminal
- Testes básicos de conectividade

## Tecnologias utilizadas

- Python
- Socket
- Threading

## Objetivo do projeto

Este projeto foi desenvolvido com finalidade educacional para aprofundamento em:

- Redes de computadores
- Programação em Python
- Segurança da Informação
- Automação
- Fundamentos de reconhecimento em redes

## Como executar

```bash
python ScanPorta.py
```
